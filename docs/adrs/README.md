# Architecture Decision Records (ADRs)

Lightweight records of significant architectural decisions.

## Format

Each ADR is a markdown file named `NNNN-short-title.md`:

```
# NNNN - Title

**Status:** accepted | superseded | deprecated
**Date:** YYYY-MM-DD

## Context
Why this decision was needed (1-3 sentences).

## Decision
What we decided (1-3 sentences).

## Consequences
What changes as a result (bullet points).
```
